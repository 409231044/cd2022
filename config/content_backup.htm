w7

程式要打的內容:

  1. git clone https://github.com/mdecourse/scarysim.git
  2. 修改 scarysim.pro, 蓋掉第 11 行, 成為 # LIBS     += -lglut -lGLU
  3. cd scarysim
  4. qmake -o Makefile scarysim.pro
  5. mingw32-make -f Makefile.Release 可以在 release 目錄取得 scarysim.exe
  6. 因為 scarysim.exe 執行需要 base.obj, arm1.obj, arm2.obj, arm3.obj 等零件外型檔案, 以及對應的 .mtl 材質檔案, 因此可將 scarysim.exe 移至倉儲根目錄執行.

第一題
利用nx12修改 "絞鍊"與 "front joint" 零件的外型, 使能將 "Triangular connecting rod", "left link" 與 "end link" 等零組件, 與 "right crank" 與 "right link" 的組件左右互換

-------------------------------------------------------------------------------

第二題
將新幾何外型組立的 uArm 機械手臂後, 將組立件轉入 CoppeliaSim 4.3.0 rev3, 以 slider 控制前三軸的轉動
-------------------------------------------------------------------------------


第三題
與同組組員合作, 在 Teams 會議中, 以畫面分享及錄影功能, 由兩位 (含) 以上組員測試題目 2. 的 XML UI slider 控制功能
-------------------------------------------------------------------------------

第四題
由兩位 (含) 以上組員協同在新幾何外型組立的 uArm 機械手臂場景中, 加入 suctionPad 零件, 以手動操控或程式操控的方式, 錄製 block 方塊零件的取放模擬影片
-------------------------------------------------------------------------------

第五題
由兩位 (含) 以上組員協同在新幾何外型組立的 uArm 機械手臂場景中, 以跨網路 Python remote API 的方式, 錄製圖形介面 slider 控制前三軸的轉動影片
